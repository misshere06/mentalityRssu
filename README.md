# Психологический Мониторинг Студентов РГСУ

Адаптивная веб-система для сбора, мониторинга и хранения информации о психологическом состоянии студентов Российского государственного социального университета.

## 🚀 Быстрый старт

### Предварительные требования

- Node.js (v14 или выше)
- npm (v6 или выше) или yarn
- Веб-браузер (Chrome, Firefox, Safari, Edge)

### Установка

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/misshere06/mentalityRssu.git
cd mentalityRssu
```

2. **Установите зависимости:**

```bash
npm install
# или
yarn install
```

3. **Скомпилируйте assets:**

```bash
npx webpack
```

4. **Запустите проект:**

Откройте `index.html` в вашем браузере или используйте локальный сервер:

```bash
npx serve
# затем откройте http://localhost:3000
```

## 🔧 Разработка

### Скрипты

- `npm install` - Установить зависимости
- `npx webpack` - Скомпилировать SCSS и JS
- `npx webpack --watch` - Режим наблюдения за изменениями
- `npx serve` - Локальный сервер для разработки

### Структура SCSS

Проект использует BEM методологию с модульной структурой:

```scss
assets/scss/
├── style.scss           # Главный файл (импортирует все компоненты)
└── components/
    ├── _variables.scss  # Переменные (цвета, шрифты, breakpoints)
    ├── _mixins.scss     # Миксины (повторяемые стили)
    ├── _base.scss       # Базовые стили
    ├── _header.scss     # Компонент Header
    ├── _sidebar.scss    # Компонент Sidebar
    ├── _cards.scss      # Компонент Cards
    ├── _tables.scss     # Компонент Tables
    ├── _forms.scss      # Компонент Forms
    ├── _charts.scss     # Компонент Charts
    ├── _widgets.scss    # Компонент Widgets
    ├── _responsive.scss # Адаптивные стили
    └── _utilities.scss  # Утилиты
```

### Адаптивный дизайн

Проект поддерживает следующие breakpoints:

- **320px**: Маленькие телефоны (iPhone 5/SE)
- **375px**: Стандартные телефоны (iPhone 6/7/8)
- **425px**: Большие телефоны (iPhone 6/7/8 Plus)
- **768px**: Планшеты (iPad)
- **1024px**: Маленькие десктопы
- **1200px**: Большие десктопы

## 🎨 Дизайн система

### Цветовая палитра

- **Основной**: `#003366` (РГСУ синий)
- **Вторичный**: `#0055A4` (Светлый синий)
- **Успех**: `#28A745` (Зеленый)
- **Предупреждение**: `#FD7E14` (Оранжевый)
- **Светлый фон**: `#F8F9FA`

### Типографика

- **Шрифт**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- **Базовый размер**: 16px (1rem)
- **Минимальный на мобильных**: 12px (0.875rem)
